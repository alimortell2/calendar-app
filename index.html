One-time setup

This app uses your own Google OAuth client so your data stays in your control.

  1. Open Google Cloud Console and create a new project (or pick an existing one).
  2. In APIs & Services → Library, enable Google Calendar API.
  3. In APIs & Services → OAuth consent screen, set User Type = External, add yourself as a test user.
  4. In Credentials → Create Credentials → OAuth client ID, choose Web application.
